## Shortcuts

| Key | Description |
| --- | --- |
| `F1` | Show Info |
| `F5` | Restart Game |
| `F11` | Toggle Fullscreen Mode |
| `F12` | DevTools |
| `Alt` + `N` | New Version (coming soons) |
| `Esc` | Close Game and Exit Window |
